# Portfolio_new

Hướng dẫn cập nhật code và đẩy lên GitHub:

1. Kiểm tra trạng thái Git:

```bash
git status --short --branch
```

2. Thêm thay đổi vào commit:

```bash
git add script.js
```

3. Tạo commit với nội dung mô tả:

```bash
git commit -m "Fix project page re-initialization when returning from summary page"
```

4. Đẩy lên GitHub:

```bash
git push
```

Nếu cần đẩy nhánh khác, thay `main` vào lệnh:

```bash
git push origin <branch-name>
```
